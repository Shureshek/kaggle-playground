# Проекты по ML & DS

В этом репозитории собраны различные учебные проекты, связанные с анализом данных и машинным обучением. Каждый проект представляет собой отдельную папку с подробным описанием и кодом.

## Содержание

| Название проекта | Описание | Стек технологий |
| --- | --- | --- |
| [aptos2019-blindness-detection](https://www.google.com/search?q=https://github.com/Shureshek/kaggle-playground/tree/main/05-aptos2019-blindness-detection) | Обнаружение и классификация стадий диабетической ретинопатии (слепоты) по снимкам сетчатки глаза. | `Python`, `PyTorch`, `ConvNext`, `OpenCV`, `Computer Vision` |
| [house-prices-advanced-regression-techniques](https://www.google.com/search?q=https://github.com/Shureshek/kaggle-playground/tree/main/04-house-prices-advanced-regression-techniques) | Предсказание цен на недвижимость (целевая переменная SalePrice) на основе большого набора характеристик зданий, участков и коммуникаций. Проект включает глубокий исследовательский анализ данных (EDA) и построение ансамбля (Stacking) из градиентных бустингов и линейных моделей. | `Python`, `Pandas`, `Seaborn`, `Scikit-learn`, `CatBoost`, `XGBoost`, `LightGBM`, `Optuna`<br> |
| [journey-to-springfield](https://www.google.com/search?q=https://github.com/Shureshek/kaggle-playground/tree/main/03-journey-to-springfield1) | Классификация жителей Спрингфилда. | `ResNet50`, `CNN`, `Albumentations`<br> |
| [digit-recognizer](https://www.google.com/search?q=https://github.com/Shureshek/kaggle-playground/tree/main/02-digit-recognizer) | Распознавание цифры в наборе данных, содержащем десятки тысяч рукописных изображений MNIST. | `Python`, `Pandas`, `Scikit-learn`, `FCNN`, `CNN`<br> |
| [titanic](https://www.google.com/search?q=https://github.com/Shureshek/kaggle-playground/tree/main/01-titanic) | Предсказание, какие пассажиры выжили после крушения «Титаника». | `Python`, `Pandas`, `Scikit-learn`, `Catboost`, `Xgboost`<br> |
