# Titanic: EDA, Baseline ML
## Обзор
Эта папка содержит исследовательский анализ данных (EDA) и базовое решение для классического соревнования Kaggle Titanic - "Machine Learning from Disaster".
Цель соревнования: Предсказать выживаемость пассажиров Титаника на основе данных о пассажирах (возраст, пол, социально-экономический класс и т.д.).

## Структура проекта
```text
01-titanic/
├── notebooks/
│   └── titanic-eda-and-baseline.ipynb  # Основной блокнот с анализом
├── data/
│   ├── train.csv                        # Обучающие данные
│   └── test.csv                         # Тестовые данные
├── README.md                            # Этот файл
└── requirements.txt                     # Зависимости Python
```

## Проведенные анализы
### 1. Исследовательский анализ данных (EDA)
- Обзор набора данных и статистика
- Анализ и обработка пропущенных значений
- Влияние признаков на выживаемость
### 2. Создание признаков (Feature Engineering)
- Извлечение титулов из имен
### 3. Модели
- Быстрая реализация с помощью Градиентного бустинга (CatBoost)
- Построение базовых моделей, объединенных Pipeline'ом c preprocessing
- **Базовые модели**:
    - Logistic Regression
    - SVM
    - Random Forest
    - XGBoost
    - CatBoost
        
- Оценка и сравнение моделей
- Настройка гиперпараметров
- Стратегия тюнинга с использованием GridSearch и кросс-валидациии
- Стратегия ансамблирования лучших моделей
